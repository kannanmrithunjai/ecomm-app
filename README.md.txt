E-commerce Website

FrontEnd Stack - React.js, JavaScritp, Axios, CSS
Backend Stack - Node.js, Express.js, REST APIs

------------------------------------------------------------------------------------------------------------------------------------------------
##Steps to RUN the Application.
NOTE: MAKE SURE Node.js in installed in your machine

Step 1: Download the Repository as Zip or Clone the Repository
Step 2: Extract the Zip folder
Step 3: Open the Project "ecomm-App" in VS Code IDE
Step 4: Open 2 terminal window [One for Frontend and another for Backend]
Step 5: move in "cd .\ecomm-frontend\" in Termnal 1
Step 6: move in "cd .\ecomm-backend\" in Terminal 2
Step 7: run "npm install" command in both the terminal
Step 8: run "npm run dev" comman in both the terminal to start the application
Step 9: Backend will be hosted in PORT:5000, Frontend will be hosted in "http://localhost:5173/"
Step 10: Open the URL: "http://localhost:5173/" in browser
